# Cell Teacher

Интерактивное веб-приложение для изучения строения и функций органелл животной клетки.
Проект создан в рамках курса Программирование на Python (2 семестр). 

##  Описание

**Cell Teacher** — это образовательное приложение, разработанное на Django, которое помогает школьникам изучать строение клетки в интерактивном формате. Приложение содержит:

- **Интерактивную схему клетки** с кликабельными органеллами
- **Подробную информацию** о каждой органелле (ядро, митохондрии, лизосомы, мембрана)
- **Режим тестирования** для проверки знаний
- **Визуально привлекательный интерфейс** с современным дизайном

## Функциональность

### Главная страница (Home)
- Интерактивная схема клетки
- Нажатие на органеллу открывает подробную информацию
- Кнопка для перехода в режим тестирования

### Страница тестирования (Test)
- Кликабельные органеллы на схеме клетки
- При нажатии на органеллу открывается окно для ввода названия
- Проверка правильности ответа

### Информационные страницы
Каждая органелла имеет свою страницу с:
- Названием органеллы (русский и английский варианты)
- Информационной картинкой с подписями
- Подробным описанием функций и особенностей


### Требования
- Python 3.10+
- Django 3.2+
